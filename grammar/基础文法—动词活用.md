# 活用分类
## 未然形（みぜんけい）
  主要用来接续表示否定的助动词“-ない”，被动、可能助动词“-れる‧られる”，使役助动词“せる‧させる”，意志、推量助动词“う”等等。  
  对应教育文法中的ない形，变化方式：  

|   |基本形   |ない形(ア段）|被动态(ア段）    |可能态      |使役态     |使役被动态   |
|:-:|:------:|:---------:|:-------------:|:---------:|:---------:|:---------:|
|五 |買う     |買**わない**  |買**われる**    |
|   |行く     |行**かない**  |行**かれる**    |
|   |泳ぐ     |泳**がない**  |泳**がれる**    |
|段 |話す     |話**さない**  |話**される**    |
|   |待つ     |待**たない**  |待**たれる**    |
|   |死ぬ     |死**なない**  |死**なれる**    |
|动 |呼ぶ     |呼**ばない**  |呼**ばれる**    |
|   |読む     |読**まない**  |読**まれる**    |
|词 |帰る     |帰**らない**  |帰**られる**    |
|   |　　     |　　　　　    |　　　　 　　    |
|一 |見る     |見**ない**    |見**られる**    |
|段 |食べる   |食べ**ない**  |食べ**られる**   |
|   |        |             |               |

|サ |する　   |**しない**　  |**される**     |
